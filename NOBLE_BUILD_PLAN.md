# Noble Hardwoods Website Rebuild Plan

## Objective

Rebuild `noblehardwoods.co` into a faster, cleaner, higher-converting local SEO website that ranks for Kansas City hardwood flooring keywords and turns visitors into quote requests, phone calls, and booked consultations.

The new site should keep Noble Hardwoods’ current premium craftsman feel: warm, relational, family-focused, and high-quality. The current brand language is strong and should be preserved while improving SEO structure, speed, mobile usability, calls to action, project proof, reviews, and lead capture.

## Primary Strategy

The current site has a strong brand foundation but needs stronger local SEO and lead generation.

The rebuild should:

1. Separate services into dedicated keyword-focused pages.
2. Build service-area pages for Kansas City suburbs.
3. Turn projects into local SEO case studies.
4. Use reviews throughout service and location pages.
5. Add strong, repeated CTAs.
6. Improve site speed and Core Web Vitals.
7. Store leads in Supabase.
8. Send lead emails through Resend.
9. Track conversions through GA4/GTM.
10. Make mobile calling and quote requests easy.

## Site Speed Plan

Target:

- Mobile PageSpeed: 85+
- Desktop PageSpeed: 95+
- Largest Contentful Paint under 2.5 seconds
- Cumulative Layout Shift under 0.1
- Low Interaction to Next Paint

Technical recommendations:

- Use Next.js or Astro.
- Host on Vercel or Netlify.
- Use static generation for service, city, blog, and project pages.
- Compress images to WebP/AVIF.
- Lazy-load gallery images.
- Avoid oversized video backgrounds.
- Use local fonts or optimized Google Fonts.
- Avoid heavy animation libraries.
- Use lightweight before/after sliders.
- Minimize third-party scripts.
- Delay non-critical scripts.
- Use image CDN.
- Preload hero image.
- Use structured data without bloating JS.
